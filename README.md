# FALTAH-Knowledge

This repository stores learning content only.

## Structure
- videos/ — MP4 files
- documents/ — PDFs and manuals
- images/ — equipment images
- equipment/ — equipment-specific folders
- data/lessons.json — lesson database read by FALTAH Enterprise

## How to add a new equipment
Edit `data/lessons.json` and write any equipment name you want. Equipment is free text.
